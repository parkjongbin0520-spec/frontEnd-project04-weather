# 👕 입을만해 (Ib-eul-man-hae)
"숫자만으로는 부족한 날씨 정보, 이제 눈으로 직접 확인하고 외출하세요."

## 📌 Project Overview
입을만해는 단순한 기온 숫자(°C)만 보고는 어떤 옷을 입어야 할지 체감하기 어려운 문제를 해결하기 위해 시작된 프론트엔드 프로젝트입니다. 실시간 기온 데이터를 분석하여 최적의 코디를 제안하고, 현재 날씨의 무드를 웹사이트 전체 테마에 투영하여 사용자에게 직관적인 경험을 제공합니다.

## 🎯 Key Target
아침마다 "오늘 뭐 입지?" 고민하며 시간을 보내는 분

숫자로 된 기온만 보고 옷을 잘못 선택해 낭패를 본 경험이 있는 분

직관적이고 감각적인 UI를 통해 빠르게 외출 준비를 마치고 싶은 분

## ✨ Core Features
1. 시각적 코디 큐레이션 (Visual Curation)
기온 데이터를 바탕으로 8단계 이상의 온도 구간을 설정하여, 각 구간에 가장 적합한 옷차림을 시각적 컴포넌트로 렌더링합니다. 단순 텍스트 나열이 아닌, 직관적인 아이콘과 이미지를 통해 한눈에 정보를 파악할 수 있습니다.

2. 기온 연동 동적 테마 (Dynamic Theme System)
현재 기온에 따라 웹사이트의 전체적인 컬러 팔레트와 배경색이 실시간으로 변경됩니다.

추운 날씨: 차가운 블루/네이비 톤의 테마

따뜻한 날씨: 포근한 옐로우/오렌지 톤의 테마

적당한 날씨: 싱그러운 그린 톤의 테마
